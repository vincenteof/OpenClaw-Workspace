## Obsidian 操作规范

### 新建笔记
默认套用 `999 - Templates/General Note.md` 模板：
```
Date: <当天日期>
Tags: 

# <标题>

# References
```

### Todo 标签
- `#active` 等状态 tag 打在 **todo 行**上，不放进 frontmatter
- 正确：`- [ ] 做某件事 #active`
- 错误：frontmatter 里写 `tags: [active]`
